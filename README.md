# 网站框架 - 快速开始指南

这是一个完整的、具有专业外观的文档网站框架，布局和风格完全基于Sharpa官网的文档页面。

## 📁 文件结构

```
website/
├── index.html           # 主HTML文件
├── styles.css           # 样式表
├── script.js            # JavaScript脚本
└── README.md            # 本文件
```

## 🎨 框架特点

✅ **完整的页面布局**
- 顶部导航栏（菜单、Logo、搜索、设置）
- 左侧导航菜单（可折叠）
- 中央主要内容区
- 底部页脚

✅ **预设的内容区块**
- 面包屑导航
- 页面标题
- 多个内容section
- 特性列表
- 规格表格
- 代码块
- 注意框
- 相关链接
- 分页导航

✅ **响应式设计**
- 自动适配桌面、平板、手机屏幕
- 移动设备上菜单可折叠

✅ **交互功能**
- 菜单折叠/展开
- 导航链接激活状态
- 搜索功能
- 设置按钮
- 分页导航

## 🚀 快速使用

### 1. 打开网站
在浏览器中打开 `index.html` 文件即可查看网站。

### 2. 替换内容

所有需要替换的内容都用 `[...]` 标记。只需找到并替换这些占位符：

#### 在 HTML 文件中 (index.html)：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `[您的公司名称]` | 网站标题 | 我的公司 |
| `[公司Logo/名称]` | 导航栏Logo | MyCompany |
| `[菜单标题]` | 左侧菜单标题 | 文档 |
| `[菜单项 1-5]` | 左侧菜单项 | 产品介绍 |
| `[Contents]` | Contents菜单标题 | 目录 |
| `[子菜单 1-4]` | 子菜单项 | 快速开始 |
| `[分类名称]` | 面包屑分类 | 产品文档 |
| `[当前页面]` | 面包屑当前页 | 产品名称 |
| `[产品/章节名称]` | 页面主标题 | Wave机械手 |
| `[小标题 1]` | 第一个section标题 | 概述 |
| `[此处放置...描述]` | 产品描述 | 详细描述文本 |
| `[关键特性]` | 特性列表标题 | 主要特性 |
| `[特性 1-5]` | 各个特性 | 自由度高 |
| `[技术规格]` | 规格表标题 | 产品规格 |
| `[参数名称 1-4]` | 规格参数名 | 重量 |
| `[参数值 1-4]` | 规格参数值 | 2.5kg |
| `[使用示例]` | 代码块标题 | 配置示例 |
| `[代码示例...]` | 代码内容 | 实际代码 |
| `[注意事项]` | 提示框标题 | 重要提示 |
| `[重要提示...]` | 提示内容 | 实际提示 |
| `[相关内容]` | 相关链接标题 | 相关文档 |
| `[相关链接 1-3]` | 相关链接 | 链接文本 |
| `[上一页]` / `[下一页]` | 分页按钮 | 按钮文本 |
| `[email@example.com]` | 邮箱地址 | sales@company.com |
| `[pr@example.com]` | 公关邮箱 | pr@company.com |
| `[链接 1-4]` | 页脚链接 | 产品 |
| `[您的公司名称]` | 页脚公司名 | MyCompany |
| `[服务条款]` / `[隐私政策]` | 法律链接 | Terms & Privacy |

### 3. 修改样式

所有样式都在 `styles.css` 中，可以根据需要修改：
- **颜色**：搜索 `#0066cc` 修改主要颜色，`#f8f8f8` 修改背景色
- **字体**：修改 `font-family` 属性
- **间距**：修改 `padding` 和 `margin` 值
- **响应式断点**：在 `@media` 查询中修改

### 4. 添加新页面

复制 `index.html` 并：
1. 修改 `<title>` 标签
2. 更新面包屑和标题
3. 替换主要内容
4. 更新菜单中的活跃链接状态

## 💡 提示

- **添加更多菜单项**：在 `<ul class="nav-list">` 中添加 `<li><a href="#">[新菜单项]</a></li>`
- **添加更多section**：复制整个 `<section>` 标签块
- **修改表格**：在 `<table>` 中添加或删除 `<tr>` 行
- **更改颜色主题**：全局替换CSS中的颜色值

## 🔧 自定义JavaScript

在 `script.js` 中可以：
- 修改 `handleSearch()` 实现实际搜索功能
- 修改 `handleSettings()` 实现设置菜单
- 修改 `handlePrevious()` 和 `handleNext()` 实现真实分页
- 添加其他交互功能
